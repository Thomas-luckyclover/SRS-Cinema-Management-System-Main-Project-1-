# 🎬 Cinema Management System — Software Requirements Specification

A complete Software Requirements Specification (SRS) for a **Cinema Management System**, covering online ticket booking, offline box office operations, F&B pre-ordering, loyalty program, and back-office management — built using the F-Soft SRS template as part of a university Business Analyst coursework project.

---

## 📋 Overview

This repository contains the full requirements documentation for a cinema ticketing platform that supports **5 main actors** (Guest, Customer, Cashier, Cinema Manager, System Admin) and **4 secondary actors** (Payment Gateway, Email/SMS Notification Service, Concession Staff, Movie Content Provider), spanning **36 use cases** across 11 functional groups.

The system covers:
- Online seat selection & booking with real-time seat locking
- Offline box office (walk-in) ticket sales
- F&B pre-ordering and concession fulfillment
- QR code check-in
- Loyalty points & voucher/promotion system
- Movie, showtime, room, and pricing management
- Review moderation and business reporting
- User account & role administration
